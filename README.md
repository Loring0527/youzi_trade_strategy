# YouZi Trade Strategy

基于「炒股养家」核心战法的 A 股量化短线策略

## 策略简介

**情绪驱动的动态低吸+追涨策略（YouZi_EmotionDriven_v2）**

- 双层判断框架：大盘趋势（宏观）× 日内情绪（微观）
- 每日计算市场情绪评分（0-100分），自动切换弱势初期/中期/末期/赚钱效应/过热高潮五阶段
- 核心指标：涨停股次日赚钱效应 + 涨停/跌停比 + 成交额趋势 + 最高连板高度
- 仓位管理：单股≤25%，最多持4只，各阶段独立仓位上限

## 快速开始

### 聚宽回测

1. 打开 [聚宽平台](https://www.joinquant.com/)
2. 新建策略，将 `jqbacktest/main.py` 全部内容粘贴进去
3. 设置回测参数：初始资金 15 万，设置回测时间
4. 运行回测

### 参数调整

策略所有可调参数集中在 `config/params.py`，修改后需同步更新 `jqbacktest/main.py` 对应参数。

## 开发工作流

本项目采用**双轨模式**：

| 文件 | 用途 | 说明 |
|------|------|------|
| `strategy/` | 本地开发 | 按职责拆分，便于阅读、调试、调参 |
| `config/params.py` | 参数管理 | 所有可调参数的权威来源 |
| `jqbacktest/main.py` | 聚宽部署 | 自包含单文件，直接粘贴到聚宽即可运行 |

**调参流程**：
1. 修改 `config/params.py` 中的参数
2. 同步更新 `jqbacktest/main.py` 顶部对应的参数字典
3. 粘贴 main.py 到聚宽运行回测验证

**注意**：`strategy/` 模块仅用于本地阅读和理解逻辑，不被 `jqbacktest/main.py` 引用（聚宽不支持外部 import）。
