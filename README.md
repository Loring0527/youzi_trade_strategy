# YouZi Trade Strategy

基于「炒股养家」核心战法的 A 股量化短线策略

## 策略简介

**情绪驱动的动态低吸+追涨策略（YouZi_EmotionDriven_v1）**

- 每日计算市场情绪指数（0-100分），自动切换低吸/追涨/空仓模式
- 核心指标：柏拉爱空指标（涨停股次日表现）+ 涨停/跌停比 + 成交额趋势 + 最高连板高度
- 仓位管理：333分仓，单股≤25%，最多持4只

## 快速开始

### 聚宽回测

1. 打开 [聚宽平台](https://www.joinquant.com/)
2. 新建策略，将 `jqbacktest/main.py` 内容粘贴进去
3. 设置回测参数：初始资金 15 万，回测时间 2019-01-01 至今
4. 运行回测

### 参数调整

策略参数在 `config/params.py` 中集中管理（本地开发用），
聚宽部署版本的参数直接在 `jqbacktest/main.py` 顶部修改。
