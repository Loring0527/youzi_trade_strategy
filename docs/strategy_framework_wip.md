# 策略框架设计（工作草稿）

> 基于炒股养家心法原文，从第一性原理重新设计。
> 本文件为探索阶段的中间记录，未最终定稿。

---

## 核心哲学

**交易的本质是群体博弈。**

> 「买入倾向的推理，大多来自赚钱效应；卖出倾向的推理，大多来自亏钱效应。」
> 「操作是瞬间的，判断是动态的。」

---

## 决策总体结构

```
第一层：市场状态判断（今天能不能做？）
    ↓ 确认赚钱效应
第二层：板块与个股选择（做什么？）
    ↓ 选出标的
第三层：仓位决策（做多重？）
    ↓ 建仓
第四层：卖点管理（什么时候走？）
```

**设计原则：第一层判断错了，后面三层做再精细也无意义。**

---

## 市场状态的三态模型

养家原文区分了三种状态（不是两种）：

| 状态 | 定义 | 操作 |
|------|------|------|
| 赚钱效应 | 短线追涨者能赚钱，热点有持续性 | 重仓追涨热点 |
| 亏钱效应 | 追涨者开始失手，但抄底者尚在撑 | **空仓** |
| 恐慌效应 | 追涨者亏，抄底者也亏，无抵抗式下跌 | 轻仓低吸（等崩溃盘宣泄殆尽） |

> 「有赚钱效应的情况下，做热点为主；有恐慌效应的情况下，做超跌为主。」

**核心设计决策：只做确定性高的行情，亏钱效应阶段坚决空仓。**
宁可错过，不做错。这与养家"重仓出击平均一年10几次"的节奏一致。

## 第一层：市场状态判断

### 每日重新判断（不保持状态）

> 「操作是瞬间的，判断是动态的。」

市场瞬息万变（突发地缘事件等），每日开盘前重新计算所有指标，不依赖昨日结论。

### 判断逻辑

```
入场条件严格，出场条件宽松：
  进入赚钱效应：三个信号全部确认 → 才操作
  退出赚钱效应：任意一个信号转差 → 立即空仓观望
```

---

### 赚钱效应判断框架（已定稿）

每日开盘前，按顺序判断：

**Step 1 — 前置条件**
```
当日全市场封板涨停数 >= 100 只
（5400只股票中有100只涨停 ≈ 1.85%，是市场有热度的最低基线）
不满足 → 直接空仓，不再计算后续指标
```

**Step 2 — 三个核心信号（全部满足才确认）**

```
指标一：打板次日赚钱效应
  取 T-2 日收盘封板的涨停股
  计算 T-1 日收-收涨幅 >= 3% 的比例
  门槛：>= 50%
  含义：超过一半的打板者次日实现有意义的盈利

指标二：炸板率
  炸板率 = 炸板数 / （封板数 + 炸板数）
  炸板定义：盘中 high >= high_limit*0.999 但 close < high_limit*0.999
  门槛：< 30%
  含义：10只冲板的至少7只能封住，资金有信心

指标三：连板存活率
  连板存活率 = T-1继续封板数 / T-2封板总数
  门槛：>= 20%
  含义：昨日涨停股中有1/5今日继续涨停，动能在延续
```

**Step 3 — 结论**
```
全部通过 → 赚钱效应确认，进入第二层（选股）
任一不通过 → 空仓观望
```

**待回测验证的参数：**
`100只 / 3% / 50% / 30% / 20%`

### 恐慌效应判断框架（已定稿）

**大盘指数定义：**
```
大盘指数日涨跌幅 = （上证指数涨跌幅 + 深证成指涨跌幅）/ 2
```
理由：游资股票池横跨沪深两市，单用上证会因大盘蓝筹权重过高而低估中小盘恐慌程度。从交易者视角，大家都盯着指数，指数本身会制造情绪效应（群体博弈的自我强化机制）。

**恐慌效应确认（进入等待模式）：**
```
大盘指数近 10 日累计跌幅 <= -15%
（不要求连续，中间出现技术性反弹小阳线也不重置计数）
```

**低吸时机（在恐慌效应已确认的前提下才触发）：**
```
当日大盘指数跌幅 <= -2%
（再来一根大阴线 = 养家原文"连续下跌一段后再出现大阴线"）
```

**原文依据：**
> 「连续下跌一段后，再出现大阴线，指数远离5日线；板块超跌后再次出现集体性大幅杀跌。当两者结合，便是超跌的出击时刻。」

**待验证参数：** `-15%（10日累计）/ -2%（单日）`

**备选方案（供后续回测对比）：**
- 方案A：全程用跌停数（>=50只的天数 >= 6/10天 + 当日跌停数放大1.5倍）
- 方案B：全程用大盘指数（当前选用）
- 方案C：混合（跌停数判断广度 + 指数判断幅度）

## 第二层：板块与个股选择（已定稿）

### 目标

赚钱效应确认后，回答两个问题：
1. 今天哪个概念是热点？（板块选择）
2. 这个热点里谁是龙头？（个股选择）

### 板块定义

使用**聚宽概念板块**（300+ 个概念），而非行业板块。理由：游资操盘以主题/题材为单位，概念板块更贴近游资视角。

### 热点板块识别

```
当日某概念板块涨停数 >= 3 → 进入热点候选池
（一只股票可属于多个概念，不影响判断）
```

### 龙头识别：早鸟频率算法

**核心定义：**
龙头 = 历史上每次这个板块启动时，第一个涨停的那只股票。

```
【离线部分：每周更新一次】

对每只股票 s，统计过去 60 天的早鸟频率：

  for 每天 t in 过去60天:
    for 每个概念 c in 股票 s 所属概念:
      if 当天 c 激活（涨停数 >= 3）AND 前2天 c 涨停数 < 3:
        这是一次"板块首发激活"事件
        if 股票 s 当天涨停:
          s 在本次激活中是"早鸟"

  早鸟频率 = s 参与的早鸟次数 / s 所属概念的总激活次数

【实时部分：每日收盘后】

1. 找当日热点概念（涨停数 >= 3 且前2天 < 3）
2. 在该概念股票池里按早鸟频率排序
3. 早鸟频率最高 + 今日涨停 → 龙头确认
4. 早鸟频率最高 + 今日未涨停但涨幅靠前 → 龙头候选（可低吸）
```

### 买入信号：低吸（v1 只做低吸）

**背景：**
低吸 = 在热点方向已确认的前提下，等强势股回调到风险收益比合理的位置买入。

追涨（打板）在量化回测中成交假设失真（实盘打板排队，回测假设立即成交），v1 先只做低吸。

**信号触发条件（基于日线数据）：**

```
1. 股票属于当日热点概念（该概念涨停数 >= 3）

2. 昨日涨停
   close[T-1] >= high_limit[T-1] × 0.999

3. 今日低开
   open[T] < close[T-1] × 0.97

4. 回调幅度在合理范围内（不能跌太深）
   open[T] > close[T-1] × 0.85
   （跌超15%说明趋势反转，不买）

5. 今日缩量（洗盘特征，非出货）
   volume[T] < volume[T-1] × 0.5
```

**洗盘 vs 出货判断依据：**

| | 洗盘（买） | 出货（不买） |
|--|--|--|
| 成交量 | 缩量 < 昨日50% | 放量 > 昨日80% |
| 跌幅 | 有限 3%-10% | 较大 > 10% |
| 含义 | 主力清洗浮筹 | 主力离场 |

**买入执行：**
聚宽日线回测默认次日开盘价成交。intraday 择时（09:30 vs 10:00）留待分钟级回测阶段优化。

## 第三层：仓位决策（已定稿）

```
单笔仓位：30%（v1 固定，不区分赚钱效应强弱）
最多同时持股：2 只
建仓方式：一次建完，不加仓（v1 简化）
```

**设计理由：**
- 养家风格是集中不分散，持 4-5 只每只仓位太小，翻倍也影响不了整体
- 加仓逻辑增加参数复杂度，v1 先跑通基础逻辑

**待后续回测验证的参数：**
`30% / 2只`

## 第四层：卖点管理（已定稿）

优先级从高到低：

```
优先级 1：第一层信号转差 → 立即清仓（不管盈亏）
  赚钱效应三指标任意一个不通过 → 全部持仓清空
  这是最强的卖出信号，市场环境变了，持仓理由消失

优先级 2：止损
  买入后跌破 -5% → 全清，不犹豫

优先级 3：浮盈保护
  浮盈曾超过 15% → 从最高点回撤 5% → 全清
  （保护利润不回吐，宁可少赚一段）

优先级 4：超时清仓
  持仓超过 10 天，盈亏在 ±2% 以内 → 全清
  （资金有机会成本，僵尸仓位要清理）
```

**待回测验证的参数：**
`-5%止损 / 15%触发浮盈保护 / 峰值-5% / 10天超时`

## 参考原文关键句

- 「买入倾向的推理，大多来自赚钱效应，卖出倾向的推理，大多来自亏钱效应。」
- 「有赚钱效应的情况下，做热点为主；有恐慌效应的情况下，做超跌为主。」

- 「重仓出击相对少些，平均一年10几次左右。」
- 「满仓出击时……胜率要求90%以上。」
- 「操作是瞬间的，判断是动态的。」
