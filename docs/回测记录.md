### 问题诊断

**问题1：过热阈值过低，牛市大量空仓**
- HOT 阈值 ≥ 70，2019年Q1牛市中有 36.4% 的时间被判为"过热"
- 过热期间不开新仓且持续减仓，错过市场从底部涨起的 37% 涨幅

**问题2：盈亏比倒置（0.78 < 1）**
- 平均盈利 ~+1.6%，平均亏损 ~-3.5%
- 赢利仓位被超时规则（持仓10天 < 2%）过早强制清仓
- 止损 -7% 绝对额远大于平均赢利

**问题3：弱势中期（WEAK_MID）从未触发**
- 趋势判断窗口仅 3 日，分数从高位快速跌破 50 时直接触发 WEAK_EARLY
- 需要扩大窗口或调整阈值

### 下一步优化方向

| 优先级 | 问题 | 优化方向 |
|--------|------|---------|
| 高 | 过热阈值 70 太低 | 提高至 80，或引入"持续 N 日"确认机制 |
| 高 | 盈亏比 0.78 | 延长赢利持仓，超时清仓改为仅针对亏损仓 |
| 中 | WEAK_MID 从未触发 | 趋势窗口 3日→5日，或调整趋势阈值 |
| 中 | 追涨选股效果待验证 | 延长回测区间，覆盖更多市场环境 |
| 低 | 回测区间仅6个月 | 延长至2019-2024多段市场验证 |
