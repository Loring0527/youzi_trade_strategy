# 策略设计文档：YouZi_EmotionDriven_v2

## 策略定位

严格对标炒股养家原文交易理念，开发适合 A 股 10-20 万小资金的量化短线策略。

> 养家核心原则：「技术图形相对次要，对于市场情绪的把握才是重点」
> 「有赚钱效应的情况下，做热点为主；有恐慌效应的情况下，做超跌为主」

---

## 核心逻辑框架

```
每日开盘前：
  1. 计算市场情绪评分（0-100）
        ↓
  2. 判断赚钱效应 / 亏钱效应
     赚钱效应（STRONG） → 追涨热点
     过热高潮（HOT）    → 只减仓
     亏钱效应（弱势）   → 分三期操作
        ↓
  3. 弱势三期（养家原文）
     弱势初期（WEAK_EARLY）→ 极轻仓低吸 + 快速止损
     弱势中期（WEAK_MID）  → 专注超跌股
     弱势末期（WEAK_LATE） → 低吸为主 + 小仓追涨
        ↓
  4. 生成卖出信号（止损优先）
        ↓
  5. 生成买入信号
        ↓
开盘执行：先卖后买
```

---

## 市场情绪评分

### 指标构成（对标养家原文描述的核心观察点）

```
情绪评分 =
  涨停打板次日赚钱效应  × 40%   ← 养家核心：打板者次日是否还能赚钱
  涨停/跌停家数比       × 30%   ← 养家：涨停多=赚钱效应，跌停多=亏钱效应
  大盘成交额趋势        × 20%   ← 养家：量能判断市场活跃度
  最高连板高度（辅助）  × 10%   ← 量化游资活跃程度
```

### 五阶段划分

| 分数区间 | 阶段 | 养家原文对应 |
|---------|------|------------|
| ≥ 70    | HOT（过热高潮） | 高潮期，分歧加大 |
| 50-70   | STRONG（赚钱效应） | 赚钱效应扩散 |
| < 50，情绪回升 | WEAK_LATE（弱势末期） | 弱势末期，布局强势 |
| < 50，情绪平稳 | WEAK_MID（弱势中期） | 持续低迷，超跌为主 |
| < 50，情绪恶化 | WEAK_EARLY（弱势初期） | 亏钱效应扩散 |

**弱势三阶段判断依据：** 当日分数 与 近3日均值 的差值
- 差值 > +3  → 弱势末期（情绪在回升）
- 差值 < -3  → 弱势初期（情绪在恶化）
- 介于两者   → 弱势中期（情绪平稳低位）

---

## 各阶段操作逻辑

### 弱势初期（WEAK_EARLY）

养家原文：刚进入弱势，亏钱效应扩散，以轻仓低吸为主，一旦判断错误快速止损。

| 参数 | 值 | 说明 |
|------|---|------|
| 总仓位上限 | 20% | 极轻仓 |
| 最多买入 | 1只 | 试探仓位 |
| 回调要求 | ≥ 20% | 更深才值得低吸 |
| 量缩要求 | < 5日均量50% | 极度缩量 |
| 止损 | -5% | 快速止损，不恋战 |

### 弱势中期（WEAK_MID）

养家原文：低迷持续企稳，专注超跌股（跌幅大、有反弹空间的个股）。

| 参数 | 值 | 说明 |
|------|---|------|
| 总仓位上限 | 40% | 轻仓操作 |
| 最多买入 | 2只 | |
| 回调要求 | ≥ 25% | 超跌标准更严 |
| 量缩要求 | < 5日均量45% | 量缩至极 |
| 止损 | -7% | 标准止损 |

### 弱势末期（WEAK_LATE）

养家原文：情绪开始回升，出现强势个股，开始布局，可小仓位追涨。

| 参数 | 值 | 说明 |
|------|---|------|
| 总仓位上限 | 60% | 开始加仓 |
| 最多买入 | 低吸2只 + 追涨1只 | 混合模式 |
| 低吸回调要求 | ≥ 15% | 要求放宽 |
| 追涨条件 | 昨涨 > 3% + 换手 > 3% | |
| 止损 | -7% | 标准止损 |

### 赚钱效应（STRONG）

养家原文："有赚钱效应的情况下，做热点为主"。

| 参数 | 值 | 说明 |
|------|---|------|
| 总仓位上限 | 80% | 重拳出击 |
| 最多买入 | 2只 | |
| 筛选条件 | 昨涨 > 3% + 换手 > 3% | |
| 止损 | -7% | 标准止损 |

### 过热高潮（HOT）

养家原文：高潮期分歧加大，不开新仓，持仓逐步减仓。

- 不买入新股
- 浮盈 > 5% 的持仓减半
- 等待情绪回落

---

## 止盈止损规则

| 规则 | 触发条件 | 操作 |
|------|---------|------|
| 硬止损（弱势初期） | 亏损 ≥ 5% | 全清（快速止损）|
| 硬止损（其他阶段） | 亏损 ≥ 7% | 全清 |
| 浮盈保护 | 盈利 > 15% 后回撤 > 7% | 全清 |
| 过热减仓 | HOT 阶段 + 浮盈 > 5% | 减半仓 |
| 超时清仓 | 持仓 10 天 + 盈亏 < 2% | 全清 |

---

## 仓位管理

- 单股仓位上限：总资产的 25%
- 最多同时持股：4 只
- 低吸初始建仓：总可用仓位的 1/3
- 追涨初始建仓：总可用仓位的 1/2

## 回测建议

### 推荐回测时段

| 时段 | 市场特征 | 预期表现 |
|------|---------|---------|
| 2014-2015 | 大牛市 + 游资活跃 | 赚钱效应阶段应有超额收益 |
| 2016-2017 | 白马股行情 | 弱势中期低吸有效性 |
| 2019-2021 | 科技股牛市 | 追涨模式有效性 |
| 2022-2023 | 震荡下行 | 弱势三阶段空仓能力 |

### 关键指标目标

- 年化收益：≥ 50%（v2 版本验证可行性）
- 最大回撤：≤ 40%
- 胜率：≥ 50%
- 盈亏比：≥ 1.5
