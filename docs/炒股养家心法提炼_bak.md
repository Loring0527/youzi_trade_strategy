量化交易系统的基础框架
一个完整的、契合养家心法的量化系统，不能像传统多因子策略那样每天固定跑全市场，而是必须以“大局观”和“市场情绪”为总开关。框架大致包含以下四个模块：
1. 大盘与情绪环境判定模块（总开关）：判断当前处于强势阶段、弱势阶段，还是崩溃边缘。
2. 动态选股池生成模块：根据第一模块输出的市场环境，分别切入不同的选股逻辑（热点龙头池或超跌反弹池）。
3. 买卖信号执行模块：根据场内场外筹码博弈的瞬间倾向，触发具体的买点与卖点。
4. 仓位与风控管理模块：基于盈亏比和系统性风险评估，动态调整仓位。

炒股养家理念在各模块中的映射
1. 市场环境与选股（大局观与情绪驱动）
- 养家理念：交易的本质是群体博弈，选股的核心在于赚钱效应与亏钱效应的演变。他极少看重基本面或纯粹的技术图形，而是紧跟市场主线。选股要有大局观，单个个股走势本身有不确定性，把整个板块一起看就更容易理解；
- 强势阶段选股：买股的逻辑是因为赚钱效应下资金不断涌入，强者恒强。量化选股应聚焦于市场热点、主流概念以及有极强号召力的龙头股。
- 弱势阶段选股：弱势市场做超跌为主。买股逻辑是因为亏钱效应扩散导致恐慌盘宣泄殆尽，少量买盘就能推动上涨（低吸）。弱势初期做强势股回调，弱势中期做连续大幅下杀的超跌股，弱势末期做有爆发力的新热点。
2. 买点逻辑（博弈与顺势）
- 养家理念：买入的瞬间是衡量潜在买入资金与场内卖出倾向的大小，前者大于后者则买入。
- 打板/追强势股买点：板上买票占了他操作的一半，但重仓买入的一刹那，必须有对后市还有3到5个涨停空间的判断。
- 左侧/低吸买点：在连续下跌一段后，再次出现集体性的大幅杀跌时出击。绝大多数下跌期的买入是等大阴线出现后，第二天的再探底时刻。
3. 卖点逻辑（忘掉成本，只看逻辑）
- 养家理念：股票操作只有买入和卖出，没有所谓的“止损”或“止盈”。
- 卖出原则：忘记自己的成本。判断卖出的标准是：假设自己现在空仓，如果你不会买入这只股，甚至选择空仓观望，那么就直接卖出。风险控制型卖出：买入机会，卖出风险（或不确定性）。当市场的发展和自己的判断不一致时，或者赚钱效应告一段落时，果断退出。
4. 仓位管理（盈亏比与舒适度）
- 养家理念：仓位管理的核心是让自己觉得舒服，并且与市场整体的交易量呈现正相关（机会多时多操作，机会少时少操作）。
- 重仓条件：决定仓位的唯一因素是风险收益比。他满仓出击的条件极其苛刻：胜率要求在90%以上，且上涨空间要看到30%-50%以上，而下跌空间控制在3%-5%以内。
5. 风控逻辑（规避系统性风险）
- 养家理念：控制回撤最重要的就是回避系统性的崩溃风险。
- 崩溃的预判：绝大部分崩溃都有前兆。当短线追涨的人接二连三失手（赚钱效应转为亏钱效应），或者逢低抄底的人开始亏钱并减少动作时，无抵抗式下跌就会发生，此时必须离场观望。行情不好时多休息，甚至空仓。
